# Learn In Path

A simple React scaffold for the Learn In Path project.

Quick start

1. Install dependencies

```powershell
cd "c:/Users/Dancing wolf/Desktop/INTERNSHIP PROJECT/learn-in-path"
npm install
```

2. Start dev server

```powershell
npm run start
```

Notes
- This repo is a minimal scaffold. You can use Vite, CRA or your preferred bundler to bootstrap further.
- Files of interest: `src/App.jsx`, `src/index.js`, `src/components/`, `src/context/`.
